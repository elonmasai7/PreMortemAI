# 3-Minute Demo Script

## 0:00-0:30 Problem
- Show dashboard.
- Say: "Most teams detect outages too late. PreMortem AI predicts likely incidents before impact grows."

## 0:30-1:10 Setup and Trust
- Open `/setup`.
- Show live Splunk, AI, MCP, and DB status.
- Emphasize: no mock data fallback.

## 1:10-2:10 Investigation Workflow
- Open `/investigations`, create new investigation for a real service.
- Open detail page and run analysis.
- Show evidence timeline, root-cause ranking, blast radius, and recommendation.

## 2:10-2:40 Human Approval
- Open `/remediation/{id}`.
- Approve or reject with note.
- Show decision audit trail.

## 2:40-3:00 Report and Value
- Open `/reports` and show generated markdown report.
- Say: "This turns Splunk telemetry into prevention-focused decisions with traceable human control."
